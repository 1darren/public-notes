## Order of Operations 

```
SELECT
  column_name ,
  AGGREGATE(column_name)
FROM
  table_name
WHERE
  boolean_conditional
GROUP BY
  pivot_columns
HAVING
  secondary_conditional
ORDER BY
  sorting_column ASC, secondary_sorting DESC
LIMIT
  top_rows_to_select
```
